# ISXRI Repository

Welcome to the **ISXRI GitHub Repository** 🎉  

Due to my inability to regularly maintain ISXRI, this project is now an **open-source, community-driven free extension**. The goal is to provide **basic functionality for EQ2 automation**, while allowing contributors to expand and improve it over time.

---

## 🔧 Purpose
ISXRI is designed for players who want straightforward, essential automation features in EQ2.  
For more advanced, premium solutions, please consider the following:

- **[OgreBot](http://www.ogregaming.com)** – A robust, feature-rich automation tool.  
- **TBA Extension** – Link will be added once provided by the author.
